# domain-tools
Extended Linux Tools for Active Directory and SSSD

## Supports

Red Hat Enterprise Linux 8
Red Hat Enterprise Linux 7
CentOS Linux 8
CentOS Linux 7

## Installation

You can run these scripts from any directory.

## Usage

For example: ./domain-join --help
